================================================================================
CÓDIGOS BASE64 DAS IMAGENS - ASSINATURA DE EMAIL
================================================================================

COMO USAR:
--------------------------------------------------------------------------------
1. Abra o arquivo .txt correspondente à imagem que deseja usar
2. Copie o 'CÓDIGO COMPLETO PARA HTML'
3. Cole diretamente no atributo src da tag <img> no HTML

EXEMPLO:
--------------------------------------------------------------------------------
<img src="data:image/png;base64,iVBORw0KG...">

ORGANIZAÇÃO DOS ARQUIVOS:
--------------------------------------------------------------------------------
• fotos_profissionais_* = Fotos de colaboradores (1400x1400)
• logos_* = Logos das empresas (upscalling)
• imagens_90x90_* = Imagens redimensionadas para 90x90px
• icones_* = Ícones (outlook, whatsapp, location, etc)
